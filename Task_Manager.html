Command-Line Task Management System

This Python program allows administrators to manage users and assign tasks, while regular users can view and update their own tasks.

Features

  • User management (add, remove, view users)
  • Task assignment and update
  • Reporting of tasks and user activity
  • File-based storage using FileManager

Usage

To run this program on your computer:

python task_manager.py
